Atur Rentang Nilai dan Predikat

Sesuaikan rentang nilai dan deskripsi predikat. Perubahan akan disimpan khusus untuk mata pelajaran ini.

Impor Nilai dari Excel

Langkah 1: Unduh Template

Unduh template untuk memastikan format data sesuai. Jangan mengubah kolom NIS dan Nama Siswa.

Langkah 2: Unggah File

Pilih file Excel yang sudah Anda isi nilainya.

Tidak ada file dipilih.

Panduan Halaman Input Nilai

Gunakan halaman ini untuk mengelola nilai siswa dengan mudah dan efisien.

⚖️ Mengatur Bobot Nilai

Di bagian kepala tabel, Anda dapat mengatur bobot untuk setiap TP (Tujuan Pembelajaran), (RR) (Rata-Rata Nilai TP), dan PAS (Penilaian Akhir Semester). Perubahan bobot akan langsung mempengaruhi perhitungan Nilai Akhir (R).

📊 Kolom Otomatis

  • (RR): Menampilkan rata-rata nilai sumatif dari semua TP berdasarkan bobotnya.
  • R: Nilai Rapor final yang dihitung dari (RR) dan PAS sesuai bobotnya.
  • Capaian Tertinggi/Terendah: Deskripsi otomatis berdasarkan nilai TP tertinggi dan terendah siswa.

⚙️ Tombol Aksi

Gunakan tombol di atas untuk Menyimpan Nilai, melakukan Impor dari Excel untuk pengisian massal, atau Mengatur Rentang Predikat (misal: "Sangat Baik" untuk nilai 90-100).
