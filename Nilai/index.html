Panduan Penggunaan

Aplikasi ini merupakan aplikasi buku nilai digital yang bisa dipergunakan oleh guru langsung tanpa melakukan instalasi. Anda tinggal menyesuaikan sesuai kebutuhan, untuk berjaga-jaga anda dapat melakukan proses backup data.

Buku Nilai ini sesuai dengan Panduan Pembelajaran dan Asesmen 2025. Cek Di Sini

GRATIS, TIDAK UNTUK DIKOMERSILKAN

1. Pengaturan Awal (Data Master)

Mulailah dengan mengubah kelas aktif di kanan atas, lalu mengisi data-data dasar. Urutan yang disarankan adalah:

  • Info Sekolah: Masukkan detail mengenai sekolah Anda.
  • Data Siswa: Tambahkan semua siswa di kelas Anda.
  • Mapel & Ekstra: Atur mata pelajaran dan kegiatan ekstrakurikuler yang ada.
  • Tujuan Pembelajaran (TP): Input TP untuk setiap mata pelajaran. Ini adalah langkah penting untuk deskripsi rapor.

2. Input Nilai

Setelah data master lengkap, Anda bisa mulai menginput nilai:

  • Input Nilai Mapel: Pilih menu "Input Nilai Mapel" lalu klik nama mata pelajaran yang muncul.
  • Input Nilai Ekstra & Koku: Lakukan hal yang sama untuk nilai ekstrakurikuler dan kokurikuler.
  • Data Pelengkap: Jangan lupa mengisi absensi dan catatan wali kelas di menu "Data Pelengkap".

3. Cetak Rapor

Jika semua nilai sudah terisi, Anda siap untuk mencetak:

  • Cetak Rapor Siswa: Untuk mencetak rapor per individu siswa.
  • Legger: Untuk mencetak leger nilai keseluruhan kelas.

4. Manajemen Data

Gunakan fitur di bagian bawah untuk mengelola data Anda:

  • Backup Data: Sangat disarankan untuk melakukan backup secara berkala. Fitur ini akan mengunduh semua data Anda. Simpan file ini di tempat yang aman.
  • Restore: Gunakan file backup untuk mengembalikan data Anda. Perhatian: Fitur ini akan menimpa semua data yang ada saat ini.
  • Reset Semua Data: Menghapus SELURUH data dan mengembalikan aplikasi ke kondisi awal. Gunakan dengan sangat hati-hati!
